# Navigation Components

Navigation components help users move through your application and understand their current location. They provide consistent navigation patterns and clear information hierarchy.

## Available Components

### Tabs
- Content organization
- Multiple tab styles
- Icon support
- Keyboard navigation

### Breadcrumbs
- Location indication
- Hierarchical navigation
- Icon support
- Responsive design

## Common Features

All navigation components include:
- Consistent styling
- Keyboard navigation
- Screen reader support
- Responsive design
- Clear visual hierarchy

## Best Practices

1. **Information Architecture**
   - Maintain clear hierarchy
   - Use consistent patterns
   - Provide clear labels
   - Consider user flow

2. **Accessibility**
   - Implement keyboard navigation
   - Use appropriate ARIA roles
   - Provide clear focus states
   - Maintain color contrast

3. **User Experience**
   - Keep navigation simple
   - Use clear labels
   - Provide visual feedback
   - Consider mobile experience

## Theming

Navigation components use the following theme variables:
- `--color-nav-active`: Active state colors
- `--color-nav-hover`: Hover state colors
- `--color-nav-text`: Text colors
- `--color-nav-border`: Border colors
- `--color-nav-background`: Background colors
