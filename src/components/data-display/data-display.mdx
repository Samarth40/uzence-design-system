# Data Display Components

Data Display components help present information to users in clear and accessible ways. They enhance the user experience by providing context and visual hierarchy to data.

## Available Components

### Tooltip
- Contextual information display
- Multiple placement options
- Keyboard and touch support
- Accessible tooltip content

### Tag
- Label and categorization
- Multiple variants and states
- Icon support
- Dismissible option

## Common Features

All data display components include:
- Consistent styling
- Responsive design
- Accessibility support
- Clear visual hierarchy
- Interactive states

## Best Practices

1. **Information Hierarchy**
   - Use appropriate components for data type
   - Maintain consistent styling
   - Provide clear visual cues
   - Consider information density

2. **Accessibility**
   - Provide text alternatives
   - Ensure keyboard navigation
   - Maintain color contrast
   - Use appropriate ARIA roles

3. **User Experience**
   - Keep content concise
   - Use appropriate animations
   - Provide clear feedback
   - Consider mobile interactions

## Theming

Data Display components use the following theme variables:
- `--color-background`: Component backgrounds
- `--color-text`: Text colors
- `--color-border`: Borders and separators
- `--shadow-tooltip`: Tooltip shadows
- `--z-index-tooltip`: Tooltip stacking
